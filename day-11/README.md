## Key exam focus (from 28 actual questions extracted)

- **Availability is NOT crypto-controlled** — Model 2 Q1 (critical distinction!)
- **Risk = threat × vulnerability** overlap — Model 2 Q2
- **DES has 16 rounds** — Model 2 Q6
- **Symmetric**: DES, 3DES — **RSA is asymmetric** — Model 2 Q7
- **Combined encryption**: secret key transmitted asymmetrically, message symmetrically — Model 2 Q8
- **PKI**: sender's private key for authentication; recipient's private key for decryption — Model 2 Q9, Q10
- **OAuth**: Authorization Server validates identity — Model 2 Q11, AAU 2015
- **Confidentiality** = unauthorized disclosure — moee Q71, AAU 2015
- **Triple DES**: 64-bit blocks, 56-bit keys, applied three times — AAU 2015
- **Spam** = unsolicited commercial email — moee Q66
- **Security estimation** → Risk management — moee Q70
- **Authorization** = permission to access resources — Model 2 Q4
- **Security mechanisms** detect, prevent, recover — Model 2 Q5
- **WannaCry NOT security approach** — AAU 2015
- **Firewall** = stop unauthorized access — AAU 2015 (also in Day 10)
