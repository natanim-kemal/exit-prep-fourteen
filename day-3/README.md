## Key exam focus

- **Time complexity analysis** — this appears in multiple questions; master nested loop analysis
- **BST traversals** — know ALL 4 traversal orders, especially inorder = sorted
- **Queue vs Stack** — LIFO vs FIFO, trick questions about "LIFO queue"
- **Binary tree definition** — 0, 1, or 2 children per node
- **Hashing/linear probing** — trace insertion sequences
- **Non-linear DS identification** — tree and graph are non-linear; stack, queue, array are linear
- **Sorting algorithm properties** — which are stable, time complexities
