## Key exam focus (from 63 actual questions extracted)

- **Process models**: Agile Scrim (leading method), Incremental (different priority modules), Waterfall (sequential, fixed upfront) — appear in ALL 3 exam files
- **UML**: behavioral vs structural; **Instance diagram is NOT behavioral** — confirmed in moee
- **Agile Manifesto**: "Following a plan over responding to change" is AGAINST agile — AAU 2015
- **Testing**: early testing saves cost, levels (unit/integration/system/acceptance), alpha vs beta, decision coverage — moee + AAU 2015
- **Requirements**: functional vs non-functional, NOT "negative stakeholders" challenge — moee + AAU 2015
- **Tuckman's model**: Forming→Storming→Norming→Performing→Adjourning — AAU 2015
- **Risk retention**: accepting risk without action — AAU 2015
- **Evolution sequence**: Change→Impact→Release→Implementation→Release — AAU 2015
- **Architecture evaluation**: via quality attributes — AAU 2015
- **CMMI**: plan-check-act — moee
- **Ethics**: don't accept work beyond competence — moee
- **Decision coverage**: 4 tests for 2 nested if-else — AAU 2015
