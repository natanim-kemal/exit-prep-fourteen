# Day 1 — Fundamentals of Programming (14 Items)

## Files in this directory

| File | Content |
|------|---------|
| `01-programming-basics.md` | Full theory coverage: variables, operators, control flow, arrays, strings, methods, exceptions, file I/O, recursion, testing, problem-solving |
| `02-trace-exercises.md` | 18 exam-style code trace problems with step-by-step solutions + common traps |
| `03-review-checklist.md` | Mastery checklist, mnemonics, formulas, Bloom's taxonomy guide, and 1-page cram sheet |

## How to use today

1. **Read** `01-programming-basics.md` (2-3 hours) — study each section thoroughly
2. **Practice** `02-trace-exercises.md` (1-2 hours) — trace each problem manually before checking answers
3. **Self-test** `03-review-checklist.md` (30 min) — tick off what you know, review weak areas
4. **Apply** — if you have time, write actual code for the concepts you studied

## Study tips

- **Don't just read** — trace code on paper, run it mentally
- **Focus on the traps** — off-by-one, prefix vs postfix, == vs .equals()
- **The exam tests Apply & Analyze** — expect to trace unfamiliar code
- **Use the model exams** — cross-reference with `../exit exam preparation/model 2024/Model 1.pdf` for real question patterns
