## Question Bank

The consolidated `question bank.txt` contains all **1,251 MCQs** with answer keys across all 13 courses. Each day's `01-*` file includes its corresponding question bank section appended as practice questions.

## How Each Day is Organized

Each day contains 4 files:

| File | Purpose |
|------|---------|
| `01-{course}-complete.md` | Full theory + practice questions from question bank |
| `02-trace-exercises.md` | Exam-style problems with step-by-step solutions |
| `03-review-checklist.md` | Review checklist + 1-page cram sheet |
| `README.md` | How to use that day's materials |

## Question Bank Sources

The consolidated `question bank.txt` was compiled and enhanced from:

- **AAU Model 2015** — ~119 questions across all subjects
- **Model moee.docx** — ~77 questions
- **Model 2.docx** — ~60+ questions
- **SPM Model.docx** — ~10 SPM questions
- **BDU Mock 4 & Mock 5**
- **AASTU Model Exam**
- **Question generator** — AI-generated questions covering all 13 courses with answer keys

## Usage

1. Start with a course's `01-{course}-complete.md` for theory and practice questions
2. Test yourself with `02-trace-exercises.md`
3. Review with `03-review-checklist.md` and the cram sheet
4. Drill with `question bank.txt` for the full 1,251-question consolidated set
5. For final prep, see [day-14/](day-14/) for mock exam
