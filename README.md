## How Each Day is Organized

Each day contains 4 files:

| File | Purpose |
|------|---------|
| `01-{course}-complete.md` | Full theory with exam Q&As embedded |
| `02-trace-exercises.md` | Exam-style problems with step-by-step solutions |
| `03-review-checklist.md` | Review checklist + 1-page cram sheet |
| `README.md` | How to use that day's materials |

## Exam Sources Referenced

- **AAU Model 2015** — ~119 questions across all subjects
- **Model moee.docx** — ~77 questions
- **Model 2.docx** — ~60+ questions
- **SPM Model.docx** — ~10 SPM questions
- **BDU Mock 4 & Mock 5**
- **AASTU Model Exam**

## Usage

1. Start with a course's `01-{course}-complete.md` for theory
2. Test yourself with `02-trace-exercises.md`
3. Review with `03-review-checklist.md` and the cram sheet
4. For final prep, see [day-14/](day-14/) for mock exam
